# HR Tool
### Project for fun

## Getting Started
### Bootstrap application

1. Clone application.

```bash
git clone https://github.com/zismailov/hr
```
3. Run setup script

```bash
bin/setup
```

4. Run test and quality suits to make sure all dependencies are satisfied and applications works correctly before making changes.

```bash
bin/ci
```

5. Run app

```bash
bin/server
```
